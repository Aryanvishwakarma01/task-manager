# Task Manager (Frontend Project)

A simple **Task Manager web application** built using **HTML, CSS, and JavaScript**.
The project includes **user authentication and task management features**, with all data stored in **LocalStorage**.

## Features

* **User Authentication**

  * Register and Login pages
  * Dynamic form validation
  * Password strength validation using Regex
  * User session stored in LocalStorage

* **Task Management**

  * Add new tasks
  * Mark tasks as **completed**
  * Delete tasks
  * Set **priority levels**
  * Set **due dates**
  * Search tasks
  * Filter tasks

* **Data Storage**

  * All users and tasks are stored in **LocalStorage**

## JavaScript Concepts Used

* Arrays and Objects
* Array methods: `map()`, `filter()`, `some()`
* DOM Manipulation
* Event Handling
* Functions
* Ternary Operator
* LocalStorage API
* Dynamic Form Validation
* Password Regex Validation

## Project Purpose

This project was built to practice **core JavaScript concepts and frontend development**, including authentication logic, dynamic validation, and interactive UI behavior without using any external libraries or frameworks.
